Story Share Web
===
1.설명
---
```
한 페이지 내 게시물을 CRUD하여 회원간 공유하는 웹서비스입니다.
```
* 기능: [feature list](https://github.com/ElfCony/story-web/wiki/feature-list)
* 일정: [To do](https://github.com/ElfCony/story-web/projects/1?)

2.환경
---
* Backend: Spring
* Frontend: Bootstrap
* Database: MySQL
* Server: CentOS 7

3.웹페이지 구성
---
* Signin
: 로그인, 회원가입 이동
* Register
: 회원가입
* Home
: 게시물 Read/Write, 내정보 이동
* Myinfo
: 내정보 Read/Update
* Read
: 게시물 상세조회, 댓글 CRUD
* Write
: 게시물 작성
