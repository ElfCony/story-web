Story Share Web
===
1.설명
---
```
한 페이지 내 게시물을 CRUD하여 회원간 공유하는 웹서비스입니다.
```
* 기능: [feature list]()
* 일정: [To do]()

2.환경
---
* <b>Backend</b>: Spring
* <b>Frontend</b>: Bootstrap
* <b>Database</b>: MySQL
* <b>Server</b>: CentOS 7

3.웹페이지 구성
---
* Signin<br>
: 로그인, 회원가입 이동
* Register<br>
: 회원가입
* Home<br>
: 게시물 Read/Write, 내정보 이동
* Myinfo<br>
: 내정보 Read/Update
* Read<br>
: 게시물 상세조회, 댓글 CRUD
* Write<br>
: 게시물 작성
